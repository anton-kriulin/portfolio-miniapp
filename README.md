## Description

<a href="http://t.me/portfolio_w3bot/tracker" target="blank">Portfolio manager</a> is a Telegram mini-app designed to help investors manage their crypto and fiat portfolios with ease.

Portfolio manager provides an effortless way to:

 * Monitor the current value of your portfolio in real-time.
 * Access up-to-date prices for crypto and fiat assets.
 * Stay informed and make data-driven investment decisions within Telegram.

## Technologies

TypeScript, React, Next.js, REST API

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# Runs the application in development mode
$ pnpm run dev

# Builds the application for production
$ pnpm run build

# Starts the Next.js server in production mode
$ pnpm run start
```
